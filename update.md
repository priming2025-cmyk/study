# Setudy UI/UX 개선 기록

> 최종 업데이트: 2026-05-22

---

## 개요

사용자 중심의 최소 클릭 원칙으로 전체 UI/UX를 전면 개선합니다.
핵심 가치: **앱 실행 → 공부 시작까지 2탭 이내**, **꿈의 도시 건설** 시각 컨셉 도입.

---

## 변경 요약

### 1. 탭 구조 (5탭 → 4탭)

| 변경 전 | 변경 후 |
|--------|--------|
| 홈 / 계획 / 공부 / 셋터디 / 기록 | **공부** / 계획 / 셋터디 / 기록 |

**홈탭 제거 이유:**
- 공부 앱에서 홈 대시보드는 불필요한 중간 단계 (1클릭 낭비)
- 스트릭·동기 요소를 공부탭 상단 미니 배너로 통합 (정보 손실 없음)
- 앱 실행 시 바로 공부탭이 보여 즉시 공부 시작 가능

**탭 인덱스 재정의:**
```
0 = 공부 (/session)
1 = 계획 (/plan)
2 = 셋터디 (/room)
3 = 기록 (/stats)
```

관련 파일:
- `lib/src/shell/app_shell.dart`
- `lib/src/app.dart`
- `lib/src/core/providers/shell_branch_index_provider.dart`

### 2. 계획탭 개선

#### 2-1. 과목 선택 개선
- 기본 과목 프리셋 유지 (국어/영어/수학/과학 등 카테고리별)
- 나만의 과목 추가: 직접 입력 후 "자주 쓰는 과목"으로 저장
- SharedPreferences에 커스텀 과목 목록 로컬 저장
- 최근 사용 과목 상단 노출 유지

#### 2-2. 목표 공부시간 — 지하철 정류장 방식
- 기존: 칩 선택 + 직접 입력 텍스트필드
- 신규: **StationTimePicker** 커스텀 위젯
  - 지하철 트랙 형태의 슬라이더
  - 주요 정류장 (30분 단위): 0 · 30 · 60 · 90 · 120 · 150 · 180 · 210 · 240분
  - 정류장 사이: 5분 단위 세밀 조정 가능
  - 현재 선택값 중앙 대형 텍스트로 강조 표시
  - 최대 4시간 (240분)

#### 2-3. 시작시간 — 시계 아이콘 버튼으로 간소화
- 기존: "시작 시각(선택)" 레이블 + OutlinedButton
- 신규: 시계 아이콘(⏰) 1개 버튼 → 탭하면 TimePicker 열림
- 시간 선택 전: 회색 아이콘 (선택 안됨 상태 명시)
- 시간 선택 후: 파란색 아이콘 + 선택된 시간 텍스트 오른쪽 표시
- X 버튼으로 취소

#### 2-4. 계획 순서 변경 (롱프레스 드래그) ✅ 채택
- `ReorderableListView`로 계획 아이템 순서 변경 지원
- 2개 이상의 계획이 있을 때 안내 텍스트 표시: "길게 누른 후 드래그해서 순서를 바꿀 수 있어요"
- 순서 변경 시 즉시 로컬 DB 업데이트
- **채택 이유**: 시험 우선순위에 따라 과목 순서를 조절하는 니즈가 명확함. 구현 비용 대비 효용이 높음.

#### 2-5. 달력 버튼 → 월별 계획 오버뷰
- AppBar 우측 상단에 📅 달력 버튼 추가
- 탭하면 월별 계획 오버뷰 바텀시트 오픈
- 오버뷰 내용:
  - 이번 달 캘린더 그리드 (계획 있는 날 점 표시)
  - 날짜 탭 → 해당 날짜로 이동
  - 각 날짜의 과목 수/목표시간 요약 툴팁

### 3. 공부탭 개선

#### 3-1. 계획이 있을 때 (1탭 시작)
- 오늘의 계획 카드 목록 표시 (과목명 + 목표시간 + 진행률 바)
- 카드를 한 번 탭 → 해당 과목으로 즉시 타이머 시작
- 완료된 과목은 취소선 + 체크 표시
- 모든 계획 완료 시 축하 배너 표시

#### 3-2. 계획이 없을 때 (퀵스타트)
- 과목 칩 선택 + 시간 칩 선택 → 시작 버튼 (2탭)
- 시작시간 없음 (공부탭 = 지금 바로 시작)

#### 3-3. 공부 중 화면
- 현행 유지: 카메라+집중도 체크 중심
- 대형 타이머 제거, 현재 얼굴 피드백 방식 유지
- 배경음 버튼 (플로팅, 우하단)

---

### 4. 셋터디탭 개선

#### 4-1. 친구 실시간 상태 표시 (소셜 존재감)
- 친구 목록 섹션 추가 (상단)
- 상태 배지:
  - 🔴 **집중 중**: 공부 세션 실행 중 → 초대 불가, 응원하기 버튼만
  - 🟢 **온라인**: 앱 접속 중 → "같이 공부하기" 초대 버튼
  - ⚫ **오프라인**: 회색, 마지막 접속 시간 표시
- 사회적 촉진 효과: 친구들이 공부하고 있으면 나도 공부하고 싶어지는 심리 활용

#### 4-2. 방 만들기 간소화
- 방 이름 자동 생성 (닉네임 + "의 공부방")
- 1탭으로 방 생성 가능 (입력 불필요)
- 방 이름 수정은 선택 사항

#### 4-3. 그룹 탐색 기능 (신규)
- "그룹 찾기" 버튼 추가
- 학교급 선택: 중학교 / 고등학교 / 대학교
- 그룹 목록 정렬 기준: **집중점수 높은 순** (추천 이유: 학습 의지가 강한 그룹에 합류하게 되어 긍정적 동기 부여 효과)
  - 추가 제안: "빈 자리 있는 순"도 탭으로 전환 가능하게 
- 그룹 카드: 그룹명, 평균 집중점수, 현재 인원/최대 인원, 이번 주 미션 진행률
- 가입 신청 → 수락 시 미션 공유 및 셋터디방 자동 생성

---

### 5. 기록탭 개선

#### 5-1. 시각적 효과 강화
- 주간 집중 그래프: `LinearProgressIndicator` → 커스텀 바 차트
- 바에 색상 그라데이션 적용 (낮은 집중 = 연한 색, 높은 집중 = 진한 색)
- 오늘 날짜 바 강조 (펄스 애니메이션)
- 스크롤 시 카드 페이드인 애니메이션

#### 5-2. 꿈의 도시 건설 컨셉 적용 (신규)
- **기록탭 상단**: 내 도시 미니어처 카드
  - 누적 블럭 수에 따라 도시 건물 단계적 언락
  - 탭하면 전체 도시 뷰 오픈
- **공부 완료 시**: 블럭 획득 애니메이션 (+N 블럭 오버레이)
- **기록탭 통계**: 건물별 완성도 시각화 (예: 도서관 = 국어 집중시간)

## 시각적 방향 — 꿈의 도시 건설 컨셉

### 컨셉 설명
공부를 하면 블럭을 받고, 그 블럭으로 자신만의 꿈의 도시를 건설합니다.
공부할수록 도시가 성장하는 시각적 피드백을 제공해 장기 동기를 유지합니다.

### 적용 위치

| 위치 | 적용 내용 |
|------|----------|
| 기록탭 상단 | 내 도시 성장 미니맵 카드 |
| 공부 완료 시 | 블럭 획득 +N 애니메이션 팝업 |
| 기록탭 통계 | 건물 언락 프로그레스 (과목별 건물) |
| 셋터디 그룹 | 그룹 도시 합산 미션 진행률 |

### 건물 언락 체계 (예시)
| 건물 | 조건 |
|------|------|
| 작은 집 | 블럭 10개 (공부 시작!) |
| 학교 | 국어+영어+수학 각 1시간 이상 |
| 도서관 | 누적 집중 10시간 |
| 공원 | 3일 연속 공부 |
| 카페 | 셋터디 5회 참여 |
| 시청 | 누적 집중 100시간 |

## 롱프레스 드래그 의견

**채택 이유:**
1. 시험이 있는 과목을 위에 올려놓고 싶은 니즈 존재
2. Flutter `ReorderableListView`로 낮은 비용으로 구현 가능
3. iOS 미리 알림/할일 앱 등 친숙한 패턴
4. 순서가 바뀌면 공부탭에서도 같은 순서로 표시되어 일관성 유지

**주의사항:**
- 드래그 핸들 아이콘(`≡`) 오른쪽에 항상 표시
- 1개 항목일 때는 드래그 핸들 숨김
