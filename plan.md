## 블럭 vs 코인 (통화 모델) · 서포터 교환 — 요약

| 이름 | 역할 |
|------|------|
| **블럭** | 앱 안 보상 통화. 집중 세션·계획 보너스·뽑기 차감 등 기존 “코인” 적립 경로 전부 블럭 단위로 동작한다. (`coin_balances.block_balance`, `coin_events.asset = 'block'`) |
| **코인** | 기프티콘·실물 보상 등 **외부 결제·교환용** 교환 코인. 서포터가 학생에게서 블럭을 줄이고 같은 수만큼 교환 코인으로 지급할 수 있다 (MVP: 1블럭→1코인 RPC `supporter_exchange_blocks_to_redeem_coins`). |

- **서포터 연결**: 과거 가족/학습 공유 카피를 “서포터 연결”로 통일. 홈에서는 1줄 진입만 두고, 상세 화면에서 학생 요약 조회 + 블럭→코인 교환 UX 제공.
- **해야 할 일(결제·스토어)** — 향후 Phase로 분리해서 진행한다.
  - [ ] 교환 코인 **유료 충전** (PG사·인앱 결제·영수증 검증).
  - [ ] 교환 코인으로 **기프티콘 발급/연동** API (외부 공급사 또는 자체 쿠폰 레저).
  - [ ] 블럭↔코인 **환율·일일 한도·수수료** 정책 및 관리 대시보드.
  - [ ] 서포터 교환 감사·플래그 이상 패턴 로깅(어뷰징 방지).
  - [ ] 교환 코인만 사용하는 **카탈로그 장바구니·정산**.

## 집중 보조(구 방해금지) — 추후 보완

- **현재:** 앱 UI에서 **방해금지 스위치는 숨김** 처리했다. (`focusDistractionModeProvider`·로컬 저장 값은 코드에 남아 있어 나중에 다시 연결 가능)
- **이유:** “채팅 숨김”에 가깝게 느껴지거나, 실제로 막고 싶은 행동(다른 앱·알림)과 맞지 않아 **기능 정의를 더 고민한 뒤** 넣는 편이 낫다고 판단.
- **추후:** OS 집중 모드 안내, 앱 백그라운드/웹 탭 이탈 기록 등 **집중 보조 체크리스트**를 정리한 뒤, 항목별로 구현·노출 순서를 정한다.
