## UI/UX 가이드라인 (고등학생/대학생 타깃, 모던/세련)

### 제품 톤앤매너
- **차분하지만 에너지 있는 학습 앱**: 과도한 귀여움/유아틱함 금지, “프리미엄 스터디 툴” 느낌
- **정보 우선순위 명확**: 오늘 계획 → 지금까지 집중시간 → 다음 액션(시작/참여)
- **낮은 인지부하**: 입력은 최소, 선택지는 3~5개 이하, 단계는 짧게

### 핵심 UI 원칙
- **한 손 조작**: 주요 CTA는 하단/엄지 영역, 스크롤 최소화
- **피드백 즉시성**: 저장/업로드/참여 상태는 항상 로딩/스낵바/상태칩으로 즉시 표시
- **상태의 시각화**: “감시”가 아니라 **세션 신뢰도(OK/UNCERTAIN)**로 표현
- **오류 친화적**: 카메라/네트워크 실패 시에도 핵심 기능이 멈추지 않도록 “대체 플로우” 제공

### UI 구조(MVP)
- 하단 탭(권장): **홈 / 계획 / 세션 / 스터디 / 통계**
- 홈은 “요약 + 빠른 시작” 역할만, 상세 편집은 각 탭에서 수행

### 비주얼 시스템
- Material 3 기반 + **Seed color 1개로 일관성 유지**
- 카드/여백 중심(12~16dp), 둥근 모서리(12dp), 과한 그림자 지양
- 라벨은 짧게, 숫자/시간은 크게(가독성)
