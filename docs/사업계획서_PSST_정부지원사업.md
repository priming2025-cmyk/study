## P — 문제 인식 (Problem)

### 1. 문제의 배경과 규모

**[시장 현황]**

국내 사교육 시장 규모는 2023년 **27조 1,000억 원** (통계청 사교육비 조사, 2024)으로 역대 최대를 기록했습니다. 같은 기간 중·고등학생의 **스마트폰 학습 앱 이용률은 78%** (교육부 교육정보화 실태조사, 2024)에 달하며, 학습 관련 앱 다운로드 수는 매년 20% 이상 증가하고 있습니다.

그러나 이 거대한 시장에는 치명적인 공백이 있습니다.

**[핵심 문제: 집중도의 블랙박스]**

현재 학습 앱(열품타·포레스트·타이머온 등)은 공통적으로 **타이머 기반**입니다. 즉, "3시간 공부했다"는 기록은 남지만 그 3시간 중 실제로 집중한 시간이 얼마인지는 알 수 없습니다.

```
현실: 학생 A가 "3시간 공부"를 기록했습니다.
      → 실제 집중 시간: 47분
      → 나머지 133분: 스마트폰 사용, 딴짓, 졸음
      → 학부모는 모릅니다. 학생 본인도 정확히 모릅니다.
```

자체 설문 조사(N=50, 중·고등학생 대상, 2025년 12월) 결과:
- **62%**: "타이머 켜놓고 스마트폰을 사용한 경험이 있다"
- **78%**: "실제 집중 시간이 기록된 시간보다 훨씬 적을 것 같다"
- **84%**: "집중도를 객관적으로 알 수 있다면 공부에 도움이 될 것"

**[문제의 3가지 차원]**

| 대상 | 문제 |
|------|------|
| 학생 | 자기 집중도를 몰라 비효율적 공부 → 성적 정체, 동기 저하 |
| 부모 | 자녀 실제 학습 여부 확인 불가 → 불신·갈등 발생 |
| 학원/선생 | 학생별 집중도 데이터 없음 → 개인화 지도 불가 |

**[기존 솔루션의 한계]**

| 기존 방법 | 한계 |
|-----------|------|
| 뇌파 측정 헤드셋 | 수십만 원대, 착용 불편, 소비자용 미적용 |
| CCTV 감시 | 프라이버시 침해, 심리적 압박, 신뢰 하락 |
| 타이머 앱 | 집중도 측정 불가, 어뷰징 쉬움 |
| 자기 보고 | 주관적·부정확, 실행 안 됨 |

### 2. 핵심 기술: AI 집중도 감지

**기술 스택:**
```
입력: 스마트폰 전면 카메라 (실시간 비디오 스트림)
     ↓
AI 처리: MediaPipe Face Mesh (468개 얼굴 랜드마크, TFLite)
     ↓
분석 지표:
  - EAR (Eye Aspect Ratio): 눈 열림 비율 → 졸음·눈감음 감지
  - Head Pose Estimation (Euler각): 머리 방향 → 시선 이탈 감지
  - Blink Rate (분당 깜빡임): 피로도·집중도 판단
     ↓
출력: 집중도 점수 0~100 + 4단계 상태 (집중/보통/이탈/졸음)
     ↓
저장: 세션 이벤트 카운트만 Supabase 저장 (영상 비저장, 개인정보 안전)
```

**기술적 우위:**
- 영상 데이터를 서버로 전송하지 않음 → 개인정보 완전 보호
- TFLite 경량 모델 → 스마트폰 배터리 소모 최소화
- 웹(WebRTC)과 앱(네이티브 카메라) 동시 지원
- 국내 상용 학습 앱 중 동등 기술 구현 사례 없음

### 1. 수익 모델 상세

**[수익 흐름 1] B2B2C — 서포터 교환 코인 충전**

```
학생 집중 공부 → 블럭 적립
        ↓
부모(서포터)가 Setudy에서 교환 코인 유료 충전
        ↓
학생에게 코인 선물 (블럭→코인 교환 승인)
        ↓
학생이 코인으로 기프티콘 수령
```

- 결제 주체: 부모 → 학생의 지불 저항 없음
- 평균 결제 단가: 1만~3만 원 (기프티콘 1~3개)
- 월 1회 충전 시 ARPU 15,000원 이상

**[수익 흐름 2] 프리미엄 구독**

| 플랜 | 가격 | 혜택 |
|------|------|------|
| 무료 | 0원 | 기본 집중도 감지, 계획, 스터디룸 |
| 프리미엄 | 월 5,000원 | AI 학습 코치, 고급 통계, 무제한 스쿼드, 프리미엄 가챠 |
| 패밀리 | 월 8,000원 | 프리미엄 + 서포터 3명 연결 + 코인 충전 10% 할인 |

**[수익 흐름 3] 학원·기관 SaaS**

- 학원별 학생 집중도 대시보드
- 수업 중 실시간 학생 집중도 모니터링
- 월 30만 원 / 기관 (50명 기준)

**Phase 3 (18개월~): B2B 확장**
